# 4세대 포켓몬 신오도감 프로그램

## 프로그램 주요 특징
### 1. GUI 디자인 및 UX 디테일
* **도감과 비슷한 테마 색상:** 포켓몬 도감을 상징하는 빨강색 계열의 상단 패널과 푸른빛의 텍스트 창을 구성했습니다.
* **알록달록한 폰트 컬러:** `JLabel` 내에 HTML 태그를 결합하여 각 포켓몬 타입 고유의 인게임 시그니처 색상을 텍스트에 정교하게 커스텀 반영했습니다.
* **자동 스크롤 업 기능:** 대량의 전체 도감(210마리) 데이터를 조회하더라도 스크롤 바가 자동으로 맨 위(No.001 모부기)로 고정되도록 `setCaretPosition(0)` 기능을 구현하여 사용자 편의성을 극대화했습니다.
* **프로그램 아이콘:** 창 모서리와 작업 표시줄에 기본 자바 아이콘 대신 포켓몬을 상징하는 포켓몬볼 아이콘이 뜨도록 프로젝트 내 자원 경로를 동적으로 바인딩했습니다.

### 2. 철저한 예외 처리 및 유효성 검사 (Robust Error Handling)
* **빈 값(Empty String) 방지 경고창:** 검색창에 아무것도 입력하지 않고 검색을 시도할 경우, 감지하여 JOptionPane의 예외 알림 팝업창을 발생시킵니다.
* **존재하지 않는 타입 필터링:** 17가지의 공식 포켓몬 타입 외에 잘못된 텍스트나 오타를 입력할 경우 경고 팝업 및 에러 메시지를 출력하여 비정상 종료를 완전 차단했습니다.

## 기능 안내

* **1번 메뉴**: 전체 도감 보기 (1번 ~ 210번 목록 전체 출력)
* **2번 메뉴**: 특정 타입별 포켓몬 나열 (단일/듀얼 타입 동시 필터링)
* **3번 메뉴**: 포켓몬 이름으로 검색 (해당 포켓몬의 도감 번호와 타입 단독 출력)
* **4번 메뉴**: 검색한 타입 리스트 파일로 저장 (메모장 파일(예: `물_type_pokemon.txt`)로 내보내기)

## 파일 구조 및 실행 방법 
> **본 프로그램은 외부 이미지 자원을 동적으로 로드하므로, 빌드 시 아래의 실행 가이드를 준수해 주셔야 에러 없이 정상 구동됩니다.**

### 실행 가이드 (중요)
1. 깃허브에서 다운로드한 `Main.java`와 `PokeBall.jpg` 파일을 사용하시는 IDE(IntelliJ 또는 Eclipse) 프로젝트의 **`src` 폴더 안**에 나란히 함께 넣어줍니다.
2. `Main.java` 코드 맨 상단의 `package project;` 선언에 맞춰, 패키지 폴더명이 **`project`**로 지정되어 있는지 확인합니다.
3. 준비가 완료되면 `Main.java`를 실행(Run)합니다.

### 포함 파일
* `Main.java` - 도감 데이터 구축 및 Swing GUI 메인 소스 코드
* `PokeBall.jpg` - 창 타이틀 및 작업표시줄 데코레이션 전용 그래픽 소스 파일
